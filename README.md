ZAUQ INDIAN CUISINE WEBSITE

A responsive multi-page restaurant website created using HTML5 and CSS3 for Zauq Indian Cuisine.

Project Overview

This website was developed to showcase the restaurant’s:

Home page promotions
About information
Full menu
Ordering section
Contact details

The website uses:

External CSS styling
Responsive web design
CSS Grid and Flexbox layouts
Media queries
Interactive hover effects
Pages Included
Page	Description
index.html	Home page with promotions
about.html	Restaurant information
menu.html	Full food menu
order.html	Ordering categories
contact us.html	Contact details and Google Map
style.css	External stylesheet
